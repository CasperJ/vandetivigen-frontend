# Vandetivigen Frontend

A React-based frontend application for the Vandetivigen project.

## Overview

This repository contains the frontend code for the Vandetivigen application. It's built using React with TypeScript and includes data visualization capabilities using D3.js, styling with Bootstrap, and other modern web technologies.

## Project Structure

- `/frontend` - Contains the main React application
  - Built with Create React App
  - Uses TypeScript for type safety
  - Includes D3.js for data visualization
  - Styled with Bootstrap

## Getting Started

To get started with development:

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

The application will be available at [http://localhost:3000](http://localhost:3000).

## Available Scripts

In the frontend directory, you can run:

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App

For more detailed information about these scripts, see the [frontend README](/frontend/README.md).
